# Requisitos do projeto

Docker Desktop 24+
Docker Compose v2
Node.js 21+ apenas para execucao local fora do Docker
npm 10+
Git

# Portas que precisam estar livres
3001
5173
5050
9000

# Variaveis principais
DB_HOST
DB_USER
DB_PASSWORD
DB_NAME
DB_PORT
PORT
NODE_ENV
VITE_API_URL

# Observacoes
Use os arquivos `backend/.env` e `frontend/.env`.
Mantenha `package-lock.json` para fixar as versoes.
No Docker, `node_modules` fica em volume nomeado.
