# GSVR

Projeto com backend Node, frontend Vite e stack Docker.

## Inicio rapido

1. Leia `requirements.md`.
2. Suba o ambiente com `docker compose up --build`.
3. Para desenvolvimento com bind mount e hot reload, use `docker compose -f docker-compose.prod.yml up --build`.

## Documentacao

- `requirements.txt`: pre-requisitos do ambiente
- `requirements.md`: setup, portas e exemplos de `.env`

## Ambiente

- Backend: `http://localhost:3001`
- Frontend: `http://localhost:5173`
- pgAdmin: `http://localhost:5050`
- Portainer: `http://localhost:9000`
